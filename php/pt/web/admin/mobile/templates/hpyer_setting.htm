签到积分:
只要签到就赠送相应的积分数，每天只能签到一次。
是否递增模式:
要开启递增模式，则需要同时设置每天递增积分、赠送积分的最大值。
递增模式下，连续签到所获得积分为：签到积分+已签到天数*每天递增积分。
例：签到积分：5，每天递增积分：1，那么第一天签到就是：5+0*1=5，第二天就是5+1*1=6，以此类推。
当达到赠送积分最大值后，则不管再连续签到几天，都只赠送积分最大值对应的积分。
